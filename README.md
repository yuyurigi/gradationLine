# gradationLine
addon : ofxGui  
  
-keyPressed  
s : screenshot  
h : gui display/hide  
c : clear the screen  
r : Reset parameters  
1 : color scheme 1  
2 : color scheme 2  
3 : color scheme 3  
4 : color scheme 4  
5 : color scheme 5  
delete, backspace : erase one vertex  

-mousePressed  
left click : add vertices  
right click : finish the shape  
click inside the circle : finish the shape (close shape)
